# Braison Swilling — Portfolio site

This folder is a **static website** you can use for job applications. It shows your IT skills, experience, and that you can build and ship something real.

## How to view it locally

1. Open the folder in File Explorer: `d:\OneDrive\Desktop\Resume\portfolio`
2. Double-click **index.html** — it will open in your default browser.

Or from the project in Cursor: right-click `index.html` → **Reveal in File Explorer** → double-click `index.html`.

## How to put it online (free options)

- **GitHub Pages** — Create a repo, upload this folder, enable Pages. Your site will be at `https://yourusername.github.io/repo-name/`.
- **Netlify** — Drag and drop the `portfolio` folder at [netlify.com/drop](https://app.netlify.com/drop). You get a link like `https://random-name.netlify.app`.
- **Vercel** — Same idea: connect the folder or repo and they give you a URL.

Once it’s live, put the link on your resume and in job applications so recruiters can see your site.

## Resume download (PDF, TXT, HTML, MD)

The Resume section converts your Markdown resume on the fly. Buttons: **PDF** (generated in browser), **TXT**, **HTML** (open in Word to save as DOCX), **MD**. For conversion to work, serve the site over HTTP and have `Braison_Swilling_Resume.md` in the parent folder or in `portfolio`.
